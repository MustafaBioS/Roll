<h1>Roll</h1>
You're a ball, Which somehow ended up in a dungeon, Your goal is to escape that dungeon,
You have a rope, which you can use to grapple/swing, Goodluck and start rolling.
<h1>Keybinds</h1>
A -> Move Left<br>
D -> Move Right<br>
Space -> Jump<br>
LMB -> Grapple/Swing<br>
P/ESC -> Pause<br>
